The easy way to integrate the Pod Marketing Analytics Portal to your website.

== Description ==
For those people who have an active Pod Marketing Analytics account, this plugin will add the website tracking script onto your website. To learn more about the Pod Marketing Analytics Portal and to get your Pod Marketing Analytics Portal account, go to http://www.jumpdemand.me.

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `PodMarketingAnalytics` and all its contents to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
